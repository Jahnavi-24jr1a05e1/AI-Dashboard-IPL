# AI-Dashboard-IPL
Interactive IPL Dashboard using Streamlit, Pandas, and Plotly

📌 Project Overview
This project is an interactive AI Dashboard built using Streamlit, Pandas, and Plotly. The dashboard analyzes IPL (Indian Premier League) match data from 2008 to 2022 and provides useful insights through charts, metrics, and interactive filters.

🎯 Objectives
- Analyze IPL match data.
- Visualize team and player performance.
- Identify trends across different IPL seasons.
- Provide an interactive dashboard experience.

📊 Dataset Information
Dataset: IPL Matches 2008–2022

Features include:

- Match ID
- City
- Date
- Season
- Team1
- Team2
- Toss Winner
- Venue
- Winning Team
- Player of the Match

Total Records: 950

## ✨ Features

### Dataset Overview
- Display first 5 rows of dataset
- Show dataset shape
- Show column names

### Data Cleaning
- Missing value analysis
- Duplicate record analysis

### KPI Metrics
- Total Matches
- Total Teams
- Total Venues
- Total Seasons

### Interactive Filter
- Season selection dropdown

### Visualizations
1. Matches Per Season
2. Top Winning Teams
3. Toss Winner Distribution
4. Top Match Venues
5. Top Player of the Match Winners

### Insights
- Team performance trends
- Venue analysis
- Player achievements
- Season-wise match distribution

## 🛠 Technologies Used

- Python
- Pandas
- Streamlit
- Plotly


## 🚀 How to Run

### Install Required Libraries

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
